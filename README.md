# 🧠 Exercícios de Lógica de Programação com VisualG

Coletânea de **140 exercícios de lógica de programação** resolvidos em **VisualG** (Portugol) — do básico ao intermediário: variáveis, operadores, estruturas condicionais, laços de repetição, vetores e matrizes.

Um registro da base de fundamentos que sustenta minha evolução como desenvolvedor.

## 🛠️ Ferramenta

- **VisualG** (pseudocódigo / Portugol)

## 📁 Organização

Cada arquivo `.alg` corresponde a um exercício, numerado e organizado por tema.
